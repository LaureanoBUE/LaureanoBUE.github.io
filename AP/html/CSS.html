Crear un archivo CSS

Para agregar código CSS a nuestra página, lo primero es crear un archivo con cualquier nombre y extensión "css".

Por ejemplo MiEstilo.css

Configurar el archivo HTML para que use al archivo CSS

Luego de tener creado el archivo CSS, le tenemos que "avisar" al HTML para que lo tome. Lo hacemos dentro del HEAD.

La linea 6 hace esa configuración, y es siempre igual, salvo por el nombre del archivo en el atributo href, que va a depender de que nombre le pongas al archivo CSS.

Código HTML: 
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title>Mi título en pestaña :D</title>
        <link rel="stylesheet" href="MiEstilo.css">
    </head>
...
...

Color

Para ponerle color a cualquier texto, se usa la propiedad "color".

En el ejemplo de abajo le ponemos color rojo a todos los elementos del tipo h1.

Código HTML: 
...
...
<body>
    <h1> Titulo primero </h1>
</body>
...
...
Código CSS 
h1 {
  color: red;
}
Resultado

Sombra de texto

Esta propiedad tiene 4 valores:

  • El primer valor indica cuanto se desplaza la sombra con respecto al texto original para los costados.
  • El segundo valor indica cuanto se desplaza la sombra con respecto al texto original para arriba o para abajo.
  • El tercer valor indica que tan difusa es la sombra.
  • El cuarto valor indica el color de la sombra.

En el ejemplo le aplicamos distintas sombras a dos títulos.

Código HTML: 
...
...
<body>
    <h1> Titulo primero </h1>
    <h3> Otro título </h3>
</body>
...
...
Código CSS 
h1 {
  text-shadow: 2px 5px 4px red;
}

h3 {
  text-shadow: 0px -7px 9px green;
}
Resultado

Titulo primero

Animación

Las propiedades:

  • animation-name: le dice al elemento como se llama la animación que tiene que usar. Nosotros elegimos los nombres de las animaciones.
  • animation-duration: le dice al elemento cuantos segundos tiene que durar la animación.
  • animation-iteration-count: le dice al elemento cuantas veces tiene que hacer la animación antes de que se detenga. Si le ponemos "infinite" lo hará para siempre!
  • animation-direction: le dice al elemento que cuando termine la animación, en vez de volverla a empezar de cero, bruzcamente, genere la animación contraria para volver al estado inicial.

Para crear una animación (para poder asignarla a un elemento mediante animation-name) tenemos que usar la parte del código que dice "@keyframes" siguiendo con el nombre que le queramos poner a la animación.

Dentro de las llaves de "from" ponemos como queremos que arranque la animación, y dentro de "to" ponemos como queremos que termine.

"from" significa "desde" en castellano.

"to" significa "hasta" en castellano.

Podemos poner la cantidad de propiedades de CSS que queramos dentro de "from" y de "to".

Código HTML: 
...
...
<body>
    <p> Soy un párrafo que cambia de color :D </p>
</body>
...
...
Código CSS 
p {
    animation-name: miAnimacion;
    animation-duration: 4s;
    animation-iteration-count: infinite;
    animation-direction: reverse;
}

@keyframes miAnimacion {
    from {
        color: red;
    }
    to {
        color: yellow;
    }
}
Resultado

Soy un párrafo que cambia de color :D
