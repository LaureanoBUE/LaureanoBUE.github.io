Charset UTF-8

Para que cualquier explorador sepa que vamos a usar el set de caracteres UTF-8, que contiene la Ñ, tildes y todos los caracteres de nuestro idioma.

<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
    </head>
    <body>

    </body>
</html>

Títulos de distinto nivel en el Body

Recordá que podes ponerlos en el orden que quieras y la cantidad que quieras de cada uno.

Código 
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title>Mi título en pestaña :D</title>
    </head>
    <body>
        <h1>Titulo de primer nivel!</h1>
        <h2>Titulo de segundo nivel!</h2>
        <h3>Titulo de tercer nivel!</h3>
        <h4>Titulo de cuarto nivel!</h4>
        <h5>Titulo de quinto nivel!</h5>
        <h6>Titulo de sexto nivel!</h6>
    </body>
</html>

Imágenes

Dentro del atributo "src" va el link a la foto. Si la foto la tenemos en el mismo lugar que el archivo HTML, entonces ponemos solo el nombre del archivo.

Código 
<img src="imagen.png">
Resultado

Línk a otra página

El link tiene 2 componentes:

  1. Enlace a donde nos lleva el link al clickearlo. (href)
  2. Texto que se ve en el link
Código 
<a href="https://www.google.com"> Hacer click para ir al Buscador </a>
Resultado Hacer click para ir al Buscador

Lista Con orden

Noten que solo se agregan números, pero no se realiza ningún tipo de cambio de orden de los ítems

Código 
<ol>
    <li>Papa</li>
    <li>Aceite</li>
    <li>Sal</li>
    <li>Queso cheddar (baba)</li>
</ol>
Resultado
  1. Papa
  2. Aceite
  3. Sal
